## 🤖 AI 协作学习

从第一天开始建立 AI 协作习惯：

### 基础 AI 协作
- **布局问题求助**："我想实现这样的布局，应该用 flex 还是 grid？"
- **代码生成辅助**：描述设计需求，让 AI 生成初始 CSS 代码
- **调试问题解决**："为什么我的居中对齐不生效？"
- **最佳实践咨询**：学习 CSS 命名规范、代码组织方式

### AI 协作重点
- **设计思维翻译**：学会向 AI 描述设计意图，生成对应代码
- **布局方案对话**：与 AI 讨论不同布局方案的优劣
- **代码优化建议**：让 AI 分析代码质量并提出改进建议
